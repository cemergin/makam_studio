Planned makam_studio-specific drills (v2; v1 ships without a formal practice mode):

- **Perde recognition** — which perde just played?
- **Makam identification** — which maqam is this seyir from?
- **Microtonal interval ear training** — Arabic ¾-tone vs Turkish koma vs equal-tempered semitone.
- **Drone-anchored intonation** — sing or play against a tanpura, with real-time pitch-error display.
- **Seyir-completion** — given the opening of a maqam's characteristic phrase, complete it.
